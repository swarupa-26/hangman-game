Hangman Game (Python)

A simple text-based Hangman game built in Python.
Players try to guess a randomly chosen word one letter at a time before running out of allowed incorrect guesses.

 Overview

This program selects a random word from a predefined list and challenges the player to guess it letter by letter.
You have 6 chances to make incorrect guesses before the game ends.

 Features

 Random word selection from a built-in word list

 Interactive letter guessing

 Tracks incorrect guesses and displays remaining attempts

 Detects repeated guesses and invalid inputs

 Victory and game-over messages

 Requirements

Python 3.7 or higher

No external libraries needed — uses only Python’s built-in modules
